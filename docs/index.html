About. Data from the City of Toronto Open Data Portal (CKAN API) and the GBFS feed at bikesharetoronto.com. Built in Python with DuckDB, pandas, GeoPandas, NetworkX, pydeck, and deck.gl. Source notebooks in notebooks/; every map here is reproducible from uv sync && uv run jupyter lab.

One data gotcha worth flagging. The End_Station_Name column in the ridership CSVs is unreliable — it often echoes Start_Station_Name rather than the true destination. All analyses here join by End_Station_Id against the GBFS station feed to recover the correct coordinates.
